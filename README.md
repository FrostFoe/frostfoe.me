## ✨ Key Features

-   **Modern Stack:** Built with the Next.js App Router for optimal performance.
-   **Multi-Author Support:** Easily manage and display posts from multiple authors.
-   **Blazing Fast:** Optimized for a perfect Google Page Speed score.
-   **Search Functionality:** Built-in search to easily find posts.
-   **Contact Form Support:** Simple integration for a functional contact page.
-   **Styled with Tailwind CSS:** A utility-first CSS framework for rapid UI development.
-   **Image Optimization:** Uses `next/image` for automatically optimized images.
-   **Content Ready:** Write your posts in Markdown (`.md`) files.

## 🚀 Tech Stack

-   [Next.js](https://nextjs.org/) (App Router)
-   [React](https://reactjs.org/)
-   [Tailwind CSS](https://tailwindcss.com/)
-   [MDX](https://mdxjs.com/) for content

## ⚙️ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/en/download/) installed on your machine (LTS version is recommended).

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-repo/frostforge.git
    cd frostforge
    ```

2.  **Install dependencies:**

    Use `npm`, `yarn`, or `pnpm` to install the project dependencies.

    ```bash
    npm install
    ```

### Running the Development Server

To start the local development server, run the following command:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. The application will automatically reload when you make changes to the code.

## 🏗️ Production Build

To create a production-ready build of your application, run:

```bash
npm run build
```

This will create an optimized build of your application in the `.next` directory.
