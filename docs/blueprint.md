# **App Name**: HelloNext

## Core Features:

- Text Display: Display 'Hello World' text on the main page.
- Customizable greeting: Use generative AI tool to decide whether to greet the user based on information it is given, such as name, or other personal characteristics.

## Style Guidelines:

- Primary color: Slate blue (#5A7B9F), providing a calm, neutral base.
- Background color: Very light grey (#F0F2F5), offering a clean, unobtrusive backdrop.
- Accent color: Pale apricot (#E9967A), for subtle, warm highlights.
- Body and headline font: 'PT Sans' sans-serif, suitable for headlines or body text.
- Simple, centered layout for easy readability.
